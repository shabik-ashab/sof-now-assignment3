# Image Editor

A Python-based image editor with GUI using Tkinter and OpenCV.

## Project Structure

```
image-editor-skeleton/
├── main.py              # Entry point
├── app.py               # Application logic (Person 2)
├── state.py             # State management (Person 2)
├── ui.py                # User interface (Person 3)
├── processors.py        # Image processing (Person 1)
├── requirements.txt     # Dependencies
└── README.md           # This file
```


## Getting Started

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Run the application:
   ```bash
   python main.py
   ```
